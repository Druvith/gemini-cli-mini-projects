# Gemini CLI Mini-Projects

This repository serves as a collection of diverse software experiments, prototypes, and mini-applications built with the assistance of the Gemini CLI agent.

The projects herein are not limited to a specific platform or technology; they may include command-line tools, web applications, simulations, or other experimental software.

## Projects

### caffeine_crash
A terminal-based economic simulation modeling the "Stagflation" trap. It demonstrates how rising fixed costs (rent) and essential prices impact discretionary spending in a micro-economy.

## Structure

Each project is contained within its own dedicated directory. Please refer to the README file inside each project folder for specific installation and usage instructions.
